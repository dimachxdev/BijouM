Sauvegarde & Import

Exportez, restaurez et importez vos données

⬇ Sauvegarde complète (JSON)

Exporte toutes les données (ventes, stock, clients, décaissements…) dans un fichier .json. Idéal pour une sauvegarde quotidienne ou un transfert vers un autre appareil.

⬆ Restauration JSON

Restaure toutes les données depuis un fichier .json exporté précédemment. Attention : remplace toutes les données actuelles.

Sauvegarde automatique fichier

Autorise l'application à écrire kayor_data.json dans un dossier de votre choix à chaque enregistrement. Ce fichier sera inclus automatiquement dans le ZIP envoyé par email. (Nécessite Google Chrome)

🖥 Planification Windows — Lundi 08h00

Trois fichiers ont été créés dans votre dossier projet pour automatiser l'envoi hebdomadaire par email.

1. config_backup.json
Remplissez votre email Gmail, email destinataire et mot de passe d'application.
2. configurer_backup.bat
Clic droit → "Exécuter en tant qu'administrateur" pour installer la tâche planifiée.
3. backup_kayor.ps1
Script exécuté automatiquement chaque lundi. Compresse le dossier et envoie par email.
Mot de passe d'application Gmail : myaccount.google.com → Sécurité → Validation en 2 étapes → Mots de passe des applications → Créer pour "Messagerie / Windows".
